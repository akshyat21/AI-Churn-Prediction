# 📉 Customer Churn Prediction with AI Retention Advice

Predict customer churn and get AI‑generated retention recommendations.

**Live Demo:** [https://ai-churn-prediction-frmgmoqwuk5ietfikapp8bd.streamlit.app/](https://ai-churn-prediction-frmgmoqwuk5ietfikapp8bd.streamlit.app/)

## 🚀 Features
- Logistic regression model (AUC 0.84, recall 0.77 at threshold 0.29)
- Interactive Streamlit dashboard
- Groq LLM (Llama 3.3) provides personalised retention advice

## 🛠️ Tech Stack
- Python, pandas, scikit‑learn
- Streamlit, joblib
- LangChain + Groq API
